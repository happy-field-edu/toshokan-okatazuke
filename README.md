# としょかん おかたづけ マスター！

小学校低学年向けの「図書館の本を正しい場所に返す」練習ができる、2フェーズの学習用Webアプリです。

- **フェーズ1**：本のラベル（請求記号）を見て、マップで正しい棚を選ぶ
- **フェーズ2**：同じ分類の中で、著者のあいうえお順・巻数順・分類番号順に並ぶ正しい隙間に差し込む
- 実在の人気図書をもとに、毎回ちがう問題を自動生成（0〜9類・絵本の全分類から出題）
- マウス／タッチ両対応、外部画像なし、**1ファイルで完結・オフライン動作**

## つかいかた
`index.html` をブラウザで開くだけです。GitHub Pages で公開すると、タブレットからURLで開けます。

教育目的での利用を想定しています。データの保存・外部送信は一切行いません。
